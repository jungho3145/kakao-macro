# 카카오톡 공지 댓글 매크로

PC 카카오톡에서 **공지사항에 정해진 시각에 자동으로 댓글이 달리도록** 도와주는 프로그램입니다. 이벤트 댓글 달기, 선착순 응모 등에 사용할 수 있습니다.

> ⚠️ 이 프로그램은 카카오톡 공식 도구가 아닙니다. 자동화된 동작은 카카오톡 약관에 영향을 받을 수 있으니 본인 책임하에 사용해 주세요.

---

## 1. 다운로드 (Windows)

지식이 없으셔도 괜찮습니다. 둘 중 편한 것 하나만 받으세요.

| 받을 파일 | 어떤 사람에게 |
|-----------|-----------|
| **`KakaoMacroSetup.exe`** | "그냥 평범하게 설치해서 쓰고 싶다" — 설치 마법사가 시작 메뉴와 바탕화면 아이콘을 자동으로 만들어 줍니다. |
| `KakaoMacro.exe` | "설치 없이 그냥 더블클릭해서 쓰고 싶다" — USB에 담아 다른 PC에서 바로 실행할 때 편합니다. |

📥 **최신 다운로드**: https://github.com/jungho3145/kakao-macro/releases/latest

### Windows에서 실행할 때 나오는 경고 (정상입니다)

다운로드한 파일을 실행하면 다음 경고가 뜰 수 있습니다.

> **Windows에서 PC를 보호했습니다.**
> Microsoft Defender SmartScreen이 인식할 수 없는 앱의 시작을 차단했습니다.

→ **"추가 정보"** 클릭 → **"실행"** 버튼이 나타나면 누르세요.

이 경고는 코드 서명 인증서(연 비용이 듭니다)가 없는 모든 개인 제작 프로그램에서 똑같이 나옵니다. 정상 동작이며, 한 번 "실행"을 누르면 이 PC에서는 다음부터 안 뜹니다.

## 2. 처음 사용 — 7단계 자세히

⏱️ 설치 + 첫 설정까지 약 5분.

### ① 카카오톡을 켜고 댓글 달 공지를 미리 띄워 둡니다

PC 카카오톡에서 댓글을 달고 싶은 방을 열고, **"공지"** 또는 압정 아이콘을 눌러 댓글 달 공지를 화면에 표시해 두세요. **댓글 입력창과 [등록] 버튼이 둘 다 보이는 상태**여야 합니다.

> 이 프로그램은 사용자가 보고 있는 카카오톡 화면을 그대로 사용합니다. 클릭할 자리를 화면에서 직접 알려주는 방식이라, 카카오톡 창 위치를 캡처 시점부터 실행 시점까지 **움직이지 마세요**.

### ② 매크로 프로그램 실행

- 인스톨러로 설치했다면: 바탕화면의 **"카카오톡 공지 댓글 매크로"** 아이콘을 더블클릭
- 단일 파일로 받았다면: `KakaoMacro.exe` 더블클릭

### ③ 1번 항목 — 댓글 입력창까지 클릭할 위치 저장하기

댓글 입력창이 이미 화면에 보이면 **그 위치 한 곳만 저장**하면 됩니다.

1. **마우스를 카카오톡 댓글 입력창 위에 올린 채 가만히 있습니다.**
2. 매크로 창의 **"3초 뒤 마우스 위치 저장"** 버튼을 클릭합니다.
3. 3초 안에 마우스를 댓글 입력창 위에 다시 정확히 가져다 놓습니다 (창 가리지 않게).
4. 3초 카운트가 끝나면 자동 저장됩니다.

> 만약 공지 화면을 매번 새로 열어야 한다면(예: 공지 아이콘 → 특정 공지 선택 → 댓글 입력창), **순서대로 여러 번** 저장하세요. 프로그램이 등록 시점에 그 순서대로 클릭합니다.

### ④ 2번 항목 — 댓글 내용 입력

등록할 댓글 내용을 텍스트박스에 적습니다. 한글·이모지 모두 가능합니다.

### ⑤ 3번 항목 — "등록" 버튼 위치 저장하기 (⭐ 매우 중요)

카카오톡 공지 댓글은 **Enter 키로는 등록되지 않습니다.** 화면의 [등록] 버튼을 정확히 눌러야 합니다. 그래서 매크로에 [등록] 버튼 좌표도 알려줘야 합니다.

1. 카카오톡 댓글 입력창에 **임시로 아무 글자나 입력**합니다 → 그래야 [등록] 버튼이 화면에 나타납니다.
2. 마우스를 그 **[등록] 버튼 위에 올립니다.**
3. 매크로 창의 **"3초 뒤 ‘등록 버튼’ 위치 저장"** 버튼을 클릭합니다.
4. 3초 안에 마우스를 [등록] 버튼 위에 다시 정확히 가져다 놓습니다.
5. 임시로 입력한 글자는 지워두세요(또는 그대로 둬도 무방 — 매크로 실행 시 새로 붙여넣기됨).

### ⑥ 4번 항목 — 등록할 시각 입력

`연-월-일 시:분:초` 형식으로 적습니다. 예: `2026-04-23 20:00:00`

> 프로그램은 인터넷의 한국 표준 시계와 자동으로 동기화하므로, **내 PC 시계가 조금 빠르거나 느려도 정확한 시각에 등록**됩니다.

### ⑦ 실행

1. **"설정 저장하기"** 클릭 — 다음번 실행을 위해 저장합니다.
2. **"서버 시간 가져오기"** 클릭 — 서버 시간과 동기화합니다 (몇 초 걸림).
3. ⭐ **먼저 "예행 연습 (실제 등록 안 함)"** 으로 한 번 돌려 보세요. 좌표가 정확한지 클릭 동작만 미리 확인할 수 있습니다.
4. 이상이 없으면 **"시작"** 버튼을 누르세요. 입력한 시각까지 기다린 뒤 자동으로 등록합니다.
5. 진행 상태 창에 카운트다운과 결과가 표시됩니다.

> 시작 버튼을 누른 뒤에는 **카카오톡 창의 위치/크기를 절대 바꾸지 마세요.** 좌표가 어긋나서 엉뚱한 곳을 클릭하게 됩니다.

## 3. 자주 묻는 질문

**Q. "예행 연습"에서 엉뚱한 곳을 클릭해요.**
A. 카카오톡 창의 위치 또는 크기가 좌표 저장 시점과 달라졌습니다. 카카오톡 창을 원래대로 두고 좌표를 다시 저장하세요.

**Q. 댓글이 입력되긴 했는데 등록이 안 돼요.**
A. ‘등록 버튼 위치’가 비어 있거나 잘못 저장되었습니다. 5번 단계를 다시 따라 해주세요. 임시 글자를 입력해 [등록] 버튼이 보이는 상태에서 캡처해야 합니다.

**Q. 한글이 깨져서 들어가요.**
A. 키보드 한/영 상태와 무관하게 동작하도록 설계되어 있습니다. 만약 깨진다면 진행 상태 창의 오류 메시지와 함께 알려주세요.

**Q. 등록 시각이 1~2초 어긋나요.**
A. ‘서버 시간 가져오기’를 시작 직전에 한 번 다시 누르세요. 또한 PC가 절전 상태로 들어가지 않게 해주세요.

**Q. 카카오톡이 화면 앞으로 안 와요.**
A. 시작 5초 전에 자동으로 카카오톡 창을 앞으로 가져옵니다. 만약 다른 전체화면 앱(게임, 동영상 등)이 위에 있으면 실패할 수 있습니다. 등록 시각 전까지는 카카오톡이 보이는 상태로 두세요.

**Q. Windows가 ‘이 프로그램은 컴퓨터에서 실행 중인 Windows를 지원하지 않는다’고 합니다.**
A. ARM 기반 Windows를 쓰고 계신 경우입니다. v1.0.1 이후 버전에서는 해결되었습니다. 최신 [Releases](https://github.com/jungho3145/kakao-macro/releases/latest)에서 다시 받아 주세요.

**Q. 제거하고 싶어요.**
A. 윈도우의 ‘설정 → 앱 → 설치된 앱’에서 ‘카카오톡 공지 댓글 매크로’를 제거하세요. 단일 exe로 받았다면 그 파일만 삭제하면 끝입니다.

## 4. 안전 사용 팁

- **첫 실행은 반드시 ‘예행 연습’으로**: 실수로 엉뚱한 곳에 댓글이 달리는 일을 막을 수 있습니다.
- **컴퓨터를 그대로 두기**: 시작 버튼을 누른 후에는 마우스를 만지지 말고, 다른 창을 앞으로 가져오지 마세요. 카카오톡이 화면 앞에 보이는 상태로 유지하세요.
- **노트북 사용자는 절전 모드 해제**: 등록 시각까지 PC가 꺼지지 않도록 ‘설정 → 시스템 → 전원 → 화면 및 절전’에서 시간을 늘리세요.
- **WiFi 안정성 확인**: 시간 동기화에 잠깐 인터넷이 필요합니다. 시작 직전에 확인해 주세요.
- **카카오톡 약관**: 자동화 행위는 약관에 따라 제재 대상이 될 수 있습니다. 이 프로그램은 학습/개인 용도로 만들어졌으며 사용 책임은 사용자에게 있습니다.

## 5. 개발자/배포자용 정보

- 동작 원리, 빌드, 기여 방법 등은 [`build/README.md`](./build/README.md) 참고.
- 최신 .exe를 직접 빌드하려면 Windows에서 `build\build.bat` 실행.
- GitHub Actions로 자동 빌드도 지원합니다 (태그 `v*` 푸시 시 자동 릴리스 생성).

### 파일 구조

```
kakao-macro/
├── KakaoMacroSetup.exe         ← 일반 사용자에게 배포할 설치 마법사 (CI 산출물)
├── KakaoMacro.exe              ← 단일 실행 파일 (CI 산출물)
├── install.bat                 ← (소스 배포) Python 자동 설치
├── run_macro.bat               ← (소스 배포) GUI 실행
├── app.py                      ← .exe용 진입점
├── run.py                      ← 개발자용 CLI 진입점
├── src/
│   ├── gui.py                  GUI
│   ├── config.py               설정 입출력
│   ├── time_sync.py            서버 시간 동기화
│   ├── scheduler.py            정밀 스케줄러
│   ├── kakao_automation.py     창 포커스·클릭·붙여넣기
│   └── position_capture.py     CLI용 좌표 캡처
├── build/                      .exe 빌드 설정
└── .github/workflows/          CI 자동 빌드
```
